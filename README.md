# 🚀 Fullstack Portfolio (React + Laravel)

A fullstack web application built using React.js (frontend) and Laravel (backend).
This project demonstrates my ability to build and integrate frontend and backend systems, including API development, database management, and user interface design.

---

## 🧩 Tech Stack

### Frontend

* React.js
* Vite
* JavaScript

### Backend

* Laravel
* PHP
* MySQL

---

## ⚙️ Features

* User interface built with React
* REST API built with Laravel
* CRUD operations
* Database integration
* API communication between frontend & backend

---

## 📂 Project Structure

```bash
frontend-react/   # React frontend
backend-laravel/  # Laravel backend
```

---

## ▶️ How to Run

### Frontend

```bash
cd frontend-react
npm install
npm run dev
```

### Backend

```bash
cd backend-laravel
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
